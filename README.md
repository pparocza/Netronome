## Install Loopback Software

- In order to route audio between JackTrip and the Netronome, you will need either Jack Audio Router (Windows) or Rogue Amoeba's Loopback (Mac)

### Windows

#### Jack Audio Router

https://jackaudio.org/downloads/

### Mac

#### Loopback
